# Customer Support UI

CSUI is a modern, open source helpdesk customer support application.

### To install and use it in your local machine:
              
- `$ git clone https://github.com/RecepCil/CustomerSupportUI.git`

- `$ cd CustomerSupportUI`

- `$ npm install`

- `$ npm run start`

<br>

### To build:

- `$ npm run build`
